# YaMusicToTelegram
YaMusicToTelegram -скрипт, который следит за треком, играющим в Яндекс.Музыке на Windows, и публикует его в Telegram-канал: обложка, артист, название, альбом, прогресс-бар с таймером. При смене трека старый пост удаляется и публикуется новый. Отдельно обрабатываются пауза (публикуется своя картинка) и полное отсутствие воспроизведения.
